# 🎓 NumPy Capstone – Student Exam Analysis

A complete data analysis project built using **Python** and **NumPy** to analyze student exam performance, discover trends, and generate insights from academic data.

---

## 📌 Project Overview

This project focuses on analyzing student examination records using NumPy-based operations and data analysis techniques. The goal is to understand student performance patterns, subject-wise trends, score distributions, and overall academic insights.

The project demonstrates practical usage of:
- NumPy arrays
- Statistical analysis
- Data preprocessing
- Data visualization
- Performance analytics

---

## 🚀 Features

- 📊 Student score analysis
- 📈 Performance statistics
- 🧮 NumPy-based calculations
- 📉 Score distribution visualization
- 🏆 Top performer identification
- 📚 Subject-wise analysis
- 🔍 Data filtering and insights
- 📋 Summary reports

---

## 🛠️ Tech Stack
